### 为什么线上导入用不了/加载缓慢



[电子书PDF文件来自Github知名项目Chinatextbook。](https://github.com/TapXWorld/ChinaTextbook/ "点击跳转") 因此“线上导入”功能会访问Github服务器。

如果你可以连接到Github服务器，但是下载时间巨长无比（还表现为“处理中”窗口动画卡顿），

那么你可以在“设置”页面中将“Github下载加速”改为[ghfast](https://ghfast.top "ghfast")。

如果你无法连接到github服务器，
**简而言之，请使用VPN。**
