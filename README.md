## 安全提示

源悦TTS不会要求你下载除
[FFMPEG](https://ffmpeg.org/ "ffmpeg官网"), 
以外的任何软件，

且永不收费、永不__主动__弹出广告。

如果你花费了多于0.01元来下载本软件，说明你被人圈了。

你需要马上找微信/支付宝官方申请退钱，然后举报那个黑心商家。

如果你发现本程序弹出了一刀999，说明你下载到盗版了。

上网搜索如何删除流氓软件，然后
[点击此处](/releases "release")下载最新官方版本。


（注：1.后续可能会添加“看广告支持开发者”功能，不点按钮就不会弹。）

(   2.后续如果将官网迁移到自有服务器的话可能会要求用户支付0.01元下载程序，
    所有收益将会用于维护服务器，若有盈余则拿去做公益。
    
  然后，[点击此处](/releases "release")永远免费对所有人开放，
  更新与官网保持一致，官网版和github版永远相同。
